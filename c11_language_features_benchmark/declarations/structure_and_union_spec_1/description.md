**Test name**: structure_and_union_spec_1

**Description:**
This code validates C11 §6.7.2.1 (Structure and union specifiers) Example 1 and demonstrates the use of anonymous
structs and anonymous unions in C11 and how they affect member access and name visibility.
The core intentional demonstration is that C allows anonymous members inside structs and shows that it is possible to
access i directly as `v1.i`.
